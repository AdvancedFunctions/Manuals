Connecting to The Internet

Setting Up the Stingray Module to Upload Code to the Microcontroller

  1. Using your phone, tablet, or computer, scan for available WiFi networks. Look for a network named “Stingray” and select it. Connect to the network using the password: 11111111.
  2. In a browser, enter the provided IP address of the Stingray.
  3. Enter the SSID of the hotspot to give the Stingray access to the internet.
